Catalysis Data Science Model

Homepage

The catalysis dataset contains several thousand experimental datapoints that represent instances of the catalytic reaction called Oxidative Coupling of Methane. The purpose of this interactive tool is to allow the user to visually understand the distribution of the catalysis dataset through a combination of customizable scatterplots and histograms.
