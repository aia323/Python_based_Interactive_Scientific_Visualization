Catalysis Data Science Model

In this visualization you will be able to go through three different sections:
The Data Exploration tab will allow you to understand the large dataset and see trends in it through scatter plots and histograms.
The Correlation Matrix tab will allow you to interact with a matrix which shows the relationship between all the different inputs in the data set.
The Regression tab will allow you to essentially build your own multivariate regression through picking your own inputs and selecting what output you want. .

How to use: Utilize the many different sliders and drop downs in these different tabs to see how the data and regression changes.
